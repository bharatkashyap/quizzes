X is an accessibility feature to help Windows users who have physical disabilities, but it is also used by others as a means to reduce repetitive strain injury (or a syndrome called the 'Emacs Pinky'). It essentially enables the user to press keys in a serial order instead of pressing multiple keys at a time. X allows the user to press and release a modifier key, and have it remain active until any other key is pressed. Microsoft first introduced X with Windows 95.
